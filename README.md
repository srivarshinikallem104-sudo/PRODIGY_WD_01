This project is a responsive landing page developed as part of the Prodigy InfoTech Web Development Internship.

## Features
- Fully responsive design
- Clean and modern UI
- Built using HTML, CSS, and JavaScript
- Compatible with desktop and mobile devices

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
The goal of this task was to practice responsive layouts, styling, and basic interactivity using core web technologies.
